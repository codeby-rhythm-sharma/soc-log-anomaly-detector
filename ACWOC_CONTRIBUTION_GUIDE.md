# ACWOC Contribution Guide

Welcome! Thank you for your interest in contributing to the SOC Log Anomaly Detector project.

This repository is beginner-friendly and suitable for first-time open-source contributors.

---

## How to Get Started

1. Fork this repository
2. Clone your forked repository
3. Create a new branch for your changes
4. Make changes related to an open issue
5. Commit your changes with a clear message
6. Push the branch to your fork
7. Open a Pull Request

---

## Contribution Rules

- Please work on one issue at a time
- Clearly mention the issue number in your PR description
- Keep code simple and readable
- Add comments where necessary
- Do not push directly to the main branch

---

## Beginner-Friendly Issues

Look for issues labeled:
- `good first issue`
- `beginner`

These issues are ideal for new contributors.

---

## Need Help?

If you are stuck or confused, feel free to ask questions in the issue comments.
We are happy to help you learn open-source contributions.
